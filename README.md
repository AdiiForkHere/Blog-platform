# 📝 Flask Blog Platform

A simple and dynamic **Blog Platform** built using **Flask (Python)** with **HTML, CSS, and JavaScript**. This application allows users to create, read, update, and delete blog posts (CRUD operations).

---

## 🚀 Features

- ✍️ Create New Blog Posts  
- 📖 Read/View All Posts  
- ✏️ Edit Existing Posts  
- ❌ Delete Posts  
- 💾 Data Storage (SQLite / JSON / In-Memory)  
- 🎨 Clean and Responsive UI  
- ⚡ Fast and Lightweight Flask Backend  

---

## 🛠️ Technologies Used

- **Python (Flask)** – Backend framework  
- **HTML5** – Structure of web pages  
- **CSS3** – Styling and layout  
- **JavaScript (Vanilla JS)** – Interactivity  
- **SQLite** (optional) – Database
